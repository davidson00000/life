## 1. 中小法律事務所向けAIリーガルエージェント導入支援サービス

- **概要**: 弁護士1〜5名規模の法律事務所に対し、契約書レビュー・判例検索・法的文書ドラフト作成を行うAIリーガルエージェントの選定・カスタマイズ・導入を支援するサービス。法律特有の正確性要件に対応したプロンプト設計と検証フローを提供する。
- **ターゲット**: 個人〜小規模法律事務所（弁護士1〜5名）、司法書士・行政書士事務所
- **収益モデル**: 初期導入支援20〜30万円/件 × 年4〜5件 = 80〜150万円。月額サポート2万円 × 年間平均3件 = 72万円。合計150〜220万円（コスト差引後100万円超）。
- **必要なAIエージェント構成**:
  - 法務ワークフロー分析エージェント: 事務所の業務フロー（契約レビュー、判例調査、書面作成等）をヒアリングシートから自動分析し、エージェント化候補を優先度付きで提案
  - プロンプト最適化エージェント: 法律用語・判例引用の正確性を担保するためのプロンプトテンプレートを自動生成し、テストケースで精度検証
  - 導入レポート生成エージェント: 導入前後の業務時間比較、ROI試算を自動集計してレポート化
- **初期コスト**: 約10〜15万円（法務ドメイン知識のリサーチ、テンプレート設計、デモ環境構築）
- **着想元ニュース**: [法AI事務エージェント「Legal Agent」／マンスリーピッチ！ スタートアップの扉 - BRIDGE](https://news.google.com/rss/articles/CBMidEFVX3lxTE5xeHhMZzQ1cXRCQkJiZThsM0VEU3JtYmp4dTJSZUEtQ3p5S0VJOU9aTGlwOWRuVmMwZHlzTWFMdVhSWlZISXJFSjFkWFVlcUlXNGhPNWRGNDBPQU5meThPS1dXakdqSUZ3WVJhby1wczdYZExf?oc=5)

## 2. 既存RAGシステム品質診断・チューニングサービス

- **概要**: 企業が構築済みのRAG（検索拡張生成）チャットボットに対して、Ragasフレームワーク等を用いた4軸評価（忠実性・回答関連性・コンテキスト精度・コンテキスト再現率）を実施し、チャンキング戦略・Rerankerモデル・埋め込みモデルの最適化提案を納品するサービス。「作ったけど精度が悪い」RAGを改善する。
- **ターゲット**: RAGを導入済みだが回答精度に不満を持つ中堅企業のDX部門、SaaS企業のカスタマーサクセス部門
- **収益モデル**: 診断＋改善提案レポート10〜20万円/件 × 月1〜2件 = 月10〜40万円。チューニング実装支援（追加）10万円/件。年間120〜250万円（コスト差引後100万円超）。
- **必要なAIエージェント構成**:
  - RAG評価エージェント: Ragasベースで4軸の定量スコアを自動計測し、ボトルネック（検索精度 vs 生成精度）を特定
  - チャンキング最適化エージェント: 7種のチャンキング手法（固定長、セマンティック、レイトチャンキング等）を対象データに自動適用し、最適パラメータを探索
  - 改善レポート生成エージェント: 評価結果・最適パラメータ・推奨Rerankerモデルを含む改善ロードマップを自動生成
- **初期コスト**: 約5〜8万円（Ragas評価パイプライン構築、ベンチマークデータセット準備）
- **着想元ニュース**:
  - [Ragasで始めるRAG評価｜4つの項目で「どこが悪いか」を特定する](https://zenn.dev/lluminai_tech/articles/85b742cd4850d6)
  - [RAGチャンキング戦略2026：7 手法の精度比較とパラメーター最適化](https://zenn.dev/0h_n0/articles/5137ee7d4dd05d)

## 3. AIエージェント対応API設計レビューサービス

- **概要**: AIエージェントが自律的にAPI を叩く時代に合わせ、既存のREST APIがエージェントフレンドリーかどうかを診断し、改善提案を行うサービス。エラーメッセージの機械可読性、レート制限の適切性、スキーマの自己記述性、認証フローのエージェント対応度などを評価する。
- **ターゲット**: API を提供するSaaS企業、プラットフォーム企業のバックエンドチーム（特にAIエージェント連携を検討中の企業）
- **収益モデル**: API診断レポート8〜15万円/件 × 月1〜2件 = 月8〜30万円。改善実装支援15万円/件 × 年4件 = 60万円。年間100〜200万円。
- **必要なAIエージェント構成**:
  - APIクローラーエージェント: OpenAPI仕様書やエンドポイント一覧を自動解析し、エージェント利用時のボトルネック（曖昧なエラーコード、不十分なスキーマ記述等）を検出
  - エージェント模擬テストエージェント: 実際にAIエージェントとしてAPIを叩き、自律的なタスク完了が可能かどうかを自動テスト・スコアリング
  - 改善提案エージェント: 検出された課題に対して、OpenAPI仕様の修正案・エラーメッセージ改善案・ドキュメント追記案を自動生成
- **初期コスト**: 約5〜10万円（APIテストフレームワーク構築、評価基準策定）
- **着想元ニュース**: [REST API設計のベストプラクティス2026: AIエージェント時代に変わったこと・変わらないこと](https://zenn.dev/hanako_tech/articles/b3d01b81de9e36)

## 4. EdTech向けAI家庭教師品質評価サービス

- **概要**: AI個別指導（家庭教師）機能を搭載するEdTechアプリに対して、教育的効果の観点からAI家庭教師の応答品質を評価するサービス。「答えを教えるだけ」でなく「理解を促進できているか」をKMP-Benchのようなフレームワークで多角的に測定し、改善提案を行う。
- **ターゲット**: AI個別指導機能を持つEdTech企業・学習アプリ開発会社、オンライン塾・予備校のAI導入担当
- **収益モデル**: 品質評価＋改善提案15〜25万円/件 × 年5〜6件 = 75〜150万円。四半期定点評価（月額5万円） × 2社 = 年間120万円。合計100〜200万円。
- **必要なAIエージェント構成**:
  - 教育的対話評価エージェント: AI家庭教師の対話ログを分析し、ソクラテス式問答・ヒント提示・段階的説明等の教育的テクニック活用度をスコアリング
  - 学習効果シミュレーションエージェント: 仮想生徒として多様な理解度レベルの質問を投げかけ、AIが学習者に応じた適応的指導ができるかテスト
  - ベンチマークレポート生成エージェント: 評価結果を競合比較・教育学的ベストプラクティスと照らし合わせたレポートを自動生成
- **初期コスト**: 約5〜10万円（教育的評価フレームワーク構築、テストシナリオ作成、教育学リサーチ）
- **着想元ニュース**: [ソルバーから家庭教師へ: KMP-Bench を使用した LLM の教育的知性の評価](https://arxiv.org/abs/2603.02775)

## 5. 中小クリニック向けAI問診・予約自動化パッケージ

- **概要**: 個人〜中小規模のクリニック（内科・歯科・皮膚科等）向けに、24時間対応のAI問診＋予約受付チャットボットを低コストで構築・導入するパッケージサービス。大病院向けの高額なAgentforce等の大規模ソリューションではなく、月額数万円で使えるスモールクリニック特化版を提供する。
- **ターゲット**: 医師1〜5名規模の個人クリニック、歯科医院、皮膚科・眼科等の専門クリニック
- **収益モデル**: 初期構築費10〜15万円 + 月額サブスク1.5万円。年12件新規 × 初期費12万円 = 144万円 + サブスク平均8件 × 1.5万円 × 12ヶ月 = 144万円。合計288万円（LLM API・インフラコスト差引後100万円超見込み）。
- **必要なAIエージェント構成**:
  - AI問診エージェント: 症状ヒアリングの対話フローを自動実行し、緊急度判定・推奨診療科の提案を行う（医療行為に該当しない範囲）
  - 予約管理エージェント: クリニックの空き枠を参照し、患者の希望に合わせた予約候補を提示・確定処理を自動実行
  - テンプレート設定エージェント: クリニックの診療科目・診療時間・よくある問い合わせから、問診フロー・FAQを自動生成し初期セットアップを効率化
- **初期コスト**: 約15〜20万円（予約API連携開発、問診テンプレート設計、医療ドメインリサーチ）
